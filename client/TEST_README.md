# Running Tests

## Prerequisites

Ensure dependencies are installed:

```bash
npm install
```

## Running Tests

Run the test suite with Vitest:

```bash
npm test
```

## Coverage

The following components are covered by tests:

- `CheckoutDialog`: Creation flow and validation.
- `OrdersPage`: List view, empty state, loading.
- `OrderDetailPage`: Detailed view, error handling, status history.
